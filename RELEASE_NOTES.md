# Megumin Lottie Zap `v1.0.0-beta.1`

Primeira release beta publica do projeto.

## Destaques

- bot executavel com `npm start`
- QR code no terminal
- comando `/addstickerlottier` para salvar bases Lottie no banco
- comando `/lotti` para listar, reenviar original e gerar animacao com imagem
- banco SQLite com presets iniciais incluidos
- README com identidade visual Megumin

## Comandos principais

```txt
/addstickerlottier nome
/lotti list
/lotti nome original
/lotti nome animacao
```

## Banco incluso

Presets iniciais:

```txt
choro
fire
grita
padrao
rindo
sono
```
